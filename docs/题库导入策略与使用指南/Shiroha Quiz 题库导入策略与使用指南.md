# Shiroha Quiz 题库导入策略与使用指南

> 本文面向普通使用者，用于说明 Shiroha Quiz 中不同题库文件应该如何导入、如何检查识别结果，以及遇到导入异常时应该怎么处理。

---

## 一、导入入口总览

Shiroha Quiz 目前主要有三类导入入口：

| 导入类型                | 使用位置   | 适合内容                                                     |
| ------------------- | ------ | -------------------------------------------------------- |
| 单文件智能导入             | 导入题库页  | 普通题库文本、Word、Excel、TXT、PDF、普通题库 JSON                      |
| 双文件导入               | 导入题库页  | 题目和答案分成两个文件的题库                                           |
| 配置 / 备份 JSON/ZIP 导入 | 设置/导出页 | 从 Shiroha Quiz 导出的全部数据备份、批量导出的题库 JSON、含图片素材的 ZIP 备份、配置备份 |

请特别注意：

> 从 Shiroha Quiz 导出的备份 JSON/ZIP，不要放进普通题库导入区解析。  
> 备份 JSON、ZIP 备份、批量题库 JSON、配置文件应到 **设置/导出 → 导入配置 / 备份 JSON/ZIP** 中导入。

---

## 二、推荐导入顺序

一般情况下，建议按照下面的顺序操作：

1. 如果题目、选项、答案都在同一个文件中，优先使用 **单文件智能导入**。
2. 如果题目和答案分成两个文件，使用 **双文件导入**。
3. 如果文件是从 Shiroha Quiz 导出的备份 JSON/ZIP 或批量题库 JSON，进入 **设置/导出 → 导入配置 / 备份 JSON/ZIP**。
4. 导入后先查看 **识别结果预览**。
5. 确认题型、答案、选项和异常提示无误后，再创建题库。

总体原则是：标准题库优先走标准解析，复杂题库才启用兜底策略。不要为了兼容混乱格式，把原本清楚的题号、选项和答案改得更复杂。

## 三、单文件智能导入

### 适合情况

单文件智能导入适合题目、选项、答案在同一个文件中的情况。

常见支持内容包括：

- TXT 文本
- Word 文档
- Excel 表格
- PDF 文件
- 普通题库 JSON
- 直接复制粘贴的题库文本
- 共用题干 / 材料题文本

### 示例格式

完整题型模板请优先参考 [标准题库格式示例](../标准题库格式示例/标准题库格式示例.md)。单文件智能导入推荐使用“分区标题 + 普通编号 + 题干 + 选项 + 答案 + 解析”的清晰结构；如果没有分区标题，或担心题型识别不稳定，可以再给单题加题型标记。

单选题：

```text
一、单选题

1. 下列说法正确的是？
A. 选项一
B. 选项二
C. 选项三
D. 选项四
答案：A
```

填空题：

```text
二、填空题

2. 依法行政要求行政机关做到____。
答案：合法行政
解析：单空填空题按文本答案保存。
```

简答题：

```text
三、简答题

3. 简述设备启动前需要检查的内容。
参考答案：检查设备状态、核对参数、确认安全防护措施，并进行安全交底。
解析：简答题答案属于文本答案，不会按选择题答案拆分。
```

Excel 表格：

```text
题号    题型    题干    A    B    C    D    答案    解析
1    单选题    安全帽的主要作用是    保护头部    装饰作用    增加重量    无实际作用    A    安全帽用于保护头部。
2    填空题    依法行政要求行政机关做到____                    合法行政    填空题答案按文本保存。
```

普通题库 JSON：

```json
{
  "name": "示例题库",
  "questions": [
    {
      "type": "single",
      "question": "安全帽的主要作用是？",
      "options": {"A": "保护头部", "B": "装饰作用", "C": "增加重量", "D": "无实际作用"},
      "answer": ["A"],
      "analysis": "安全帽用于保护头部。"
    }
  ]
}
```

说明：普通题库 JSON 可以作为题库内容导入；Shiroha Quiz 导出的完整备份 JSON/ZIP 不属于普通题库文本，应走设置/导出页的备份恢复入口。

系统会自动尝试识别：

- 题号
- 题干
- 选项
- 答案
- 题型
- 答案解析区
- 共用题干 / 材料题上下文

### 共用题干 / 材料题建议写法

如果一段材料对应多道题，建议让材料引导行单独成行。材料题比普通题更容易把材料、引导语和小题题干混在一起，所以小题建议显式写题型标记：

```text
根据以下材料，回答 1-2 题：
这里是一段材料文本。

1. 单选题：下列说法正确的是？
A. 选项一
B. 选项二
答案：A

2. 多选题：下列做法正确的有？
A. 选项一
B. 选项二
答案：AB
```

不建议把材料引导语和正式题干挤在同一行，例如 `1. 根据材料回答1-2题，下列说法正确的是？`。这种写法会优先按普通题目处理，避免误删题干。

识别完成后，题目会进入 **识别结果预览**。请先核对结果，再确认导入。

## 四、双文件导入

### 适合情况

双文件导入适合题目和答案分离的题库。

例如：

- 一个文件只包含题目和选项
- 另一个文件单独列出答案

### 题目文件示例

```text
1. 下列说法正确的是？
A. 选项一
B. 选项二
C. 选项三
D. 选项四

2. 以下哪些属于常见题型？
A. 单选题
B. 多选题
C. 判断题
D. 简答题
```

### 答案文件示例

```text
1. A
2. ABC
```

### 匹配方式说明

| 匹配方式   | 适合情况               |
| ------ | ------------------ |
| 自动匹配   | 推荐默认使用，适合大多数题库     |
| 按题号匹配  | 适合题号清晰、答案也带题号的文件   |
| 严格题号匹配 | 适合题号必须一一对应的情况      |
| 按顺序匹配  | 适合答案顺序与题目顺序完全一致的情况 |

如果题库按章节、分卷或题型分区，并且每个分区题号会从 1 重新开始，建议题目文件和答案文件都保留相同的分区标题。

推荐：

```text
第一章 基础知识
1. A
2. B

第二章 操作规程
1. C
2. D
```

不推荐只写：

```text
1. A
2. B
1. C
2. D
```

后者存在重复题号，系统仍会尝试自动匹配，但更依赖顺序和兜底评分，导入后必须重点核对。

双文件导入后同样需要查看识别结果预览，确认答案是否正确匹配到题目。

## 五、配置 / 备份 JSON/ZIP 导入

### 使用位置

```text
设置/导出 → 导入配置 / 备份 JSON/ZIP
```

### 适合内容

该入口适合导入：

- Shiroha Quiz 导出的全部数据备份
- 题库管理页批量导出的题库 JSON
- 从其他设备迁移过来的 Shiroha Quiz 数据
- 配置备份文件

### 不适合内容

该入口不适合导入普通题库文件，例如：

- 普通 Word 题库
- 普通 TXT 题库
- 普通 PDF 题库
- 未经过 Shiroha Quiz 导出的普通文本题库

这些内容应使用 **导入题库页** 的普通题库导入功能。

### 合并导入与覆盖恢复

| 模式   | 含义                     | 适合场景             |
| ---- | ---------------------- | ---------------- |
| 合并导入 | 保留当前数据，将 JSON 内的题库追加进来 | 导入别人分享的题库、导入批量题库 |
| 覆盖恢复 | 用备份文件替换当前本地数据          | 换设备恢复、恢复完整备份     |

请注意：

> 覆盖恢复会替换当前本地数据。  
> 执行覆盖恢复前，建议先导出一份当前全部数据备份。

## 六、识别结果预览

自动识别题库后，请不要立刻确认导入，建议先查看识别结果预览。

重点检查：

- 题型是否正确
- 答案是否正确
- 选项是否完整
- 是否有“待确认”或异常提示
- 题库名称是否正确
- 题目数量是否符合预期

如果发现异常，可以先修改原始文本，再重新识别。

## 七、常见问题与处理建议

| 问题            | 可能原因                                  | 处理建议                                                                                                                                                    |
| ------------- | ------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 识别出的题目数量太少    | 文件格式复杂、题号不清晰、排版混乱                     | 尝试复制纯文本导入，或先整理题库格式                                                                                                                                      |
| 答案没有识别出来      | 答案格式不统一，或答案与题目距离太远                    | 尽量统一为“答案：A”“参考答案：ABC”等格式                                                                                                                                |
| 单选题被识别成判断题    | 选项缺失，或 A/B/C/D 格式不完整                  | 检查选项是否完整，题干和选项是否分行清楚                                                                                                                                    |
| 多选题答案错误       | 多选答案顺序或格式不统一                          | 建议写成“答案：ABC”或“答案：A、B、C”                                                                                                                                 |
| 简答/填空答案被拆散    | 把文本答案误按选择题答案清洗                        | 主观题答案应保留原文，例如“检查设备、核对参数、进行安全交底。”不要拆成 A/B/C/D 或多条客观答案                                                                                                    |
| PDF 识别不稳定     | PDF 是扫描图，或排版过于复杂                      | 优先转成 Word / TXT，或复制可选中文本后导入                                                                                                                             |
| JSON/ZIP 导入失败 | 把 Shiroha Quiz 备份 JSON/ZIP 放进了普通题库导入区 | 到“设置/导出 → 导入配置 / 备份 JSON/ZIP”中导入                                                                                                                        |
| 手机端导出没反应      | 移动端浏览器或系统文件保存能力可能受限                   | 使用“复制备份文本”或 ZIP/JSON 文件备份作为兜底方式                                                                                                                         |
| 原题库非常杂乱       | 题号、答案、题型、选项格式长期不统一                    | 可以先做格式清洗再导入；国内用户可优先使用 DeepSeek、GLM / 智谱清言，能稳定使用海外模型时也推荐 ChatGPT、Claude。隐私风险主要来自“上传到外部 AI 工具清洗”，不是 Shiroha Quiz 本地导入本身。清洗时只统一格式，不改题意、不编答案，无法确认的答案标为“待确认” |

## 八、推荐的题库整理格式

为了提高识别准确率，建议尽量整理成下面的结构。

### 单选题

```text
1. 下列说法正确的是？
A. 选项一
B. 选项二
C. 选项三
D. 选项四
答案：A
```

### 多选题

```text
2. 以下哪些属于常见题型？
A. 单选题
B. 多选题
C. 判断题
D. 简答题
答案：ABCD
```

### 判断题

```text
3. Shiroha Quiz 支持题库导入。
答案：正确
```

也可以写成：

```text
3. Shiroha Quiz 支持题库导入。
答案：对
```

### 填空题

```text
4. Shiroha Quiz 的题库数据主要保存在____中。
答案：本地存储
```

### 简答题

```text
5. 请简述导入题库前为什么要查看识别结果预览。
参考答案：因为自动识别可能存在题型、答案或选项识别错误，导入前预览可以减少错误题目进入题库。
```

## 九、移动端使用建议

手机端适合：

- 日常刷题
- 错题复习
- 查看题库
- 导入小型题库
- 导出或复制备份文本

如果题库文件很大、格式很复杂，建议优先在电脑 Web 端整理和导入。

原因是：

- 手机端文件选择不如电脑方便
- PDF 和 Word 解析可能更慢
- 大题库处理可能导致页面卡顿
- 移动端浏览器或系统文件保存能力可能受设备环境影响

---

## 十、数据安全提醒

Shiroha Quiz 的题库数据主要保存在本地浏览器或原生 App 本地存储中。

以下操作可能导致数据丢失：

- 清理浏览器缓存
- 更换浏览器
- 卸载 App
- 清理 App 数据
- 更换设备

建议：

1. 重要题库导入后，及时到 **设置/导出** 导出全部数据 JSON/ZIP。
2. 大量刷题后，定期备份全部数据。
3. 换设备前，先导出全部数据备份，再在新设备中恢复。

---

## 十一、快速判断

如果不确定该用哪个入口，可以按下面判断：

```text
普通题库文本、Word、Excel、TXT、文字层 PDF → 导入题库页
普通题库 JSON、单题库 JSON、题目数组 JSON → 导入题库页
题目和答案分开 → 导入题库页的双文件导入
Shiroha Quiz 导出的 JSON/ZIP → 设置/导出页
批量导出的题库 JSON → 设置/导出页
想恢复全部数据 → 设置/导出页
```

最简单的原则是：

> 普通题库走“导入题库”。  
> Shiroha Quiz 导出的 JSON/ZIP 走“设置/导出”。  
> 导入后先看预览，再确认创建题库。
