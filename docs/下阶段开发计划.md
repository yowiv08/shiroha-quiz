# 下阶段开发计划

当前版本：原生 `v0.4.3-native`，Web `v0.4.5-alpha`。两端核心功能已对齐。

## 已完成（v0.4.x）

- 双图标系统统一至 native 源集
- Shiroha 模式（开屏图/插画/图标统一开关）
- 题库 JSON 相互导入，含图片完全互通（ZIP + base64）
- 导入编辑器查找替换（正则）
- 填空题识别优化（关键词扩展，减少简答题误判）
- 核对页筛选器按需显示 + 自动聚焦
- 单题编辑页 + AI 建议前置
- 素材全面 WebP + 暗夜模式无边框
- Excel 表格导入 + 数学/化学符号识别
- AI 核对与解析（OpenAI 兼容）
- 暗夜/浅色模式切换 + Design Token 系统
- R8 + 资源收缩 + 正式签名
- 练习/考试主流程稳定（答题、着色、收卡、滑动切题、答题卡）

## P1：导入体验深挖

- 深化 Excel 表格导入的列映射灵活性（混合布局、合并单元格）
- 继续核校真实题库导入结果，沉淀更多解析样例
- 优化大题库（500+ 题）导入预览性能
- 导入预览异常自动批量修复建议

## P2：刷题体验增强

- 填空题答案归一化（等价答案 / 多表述匹配）
- 简答题手动评分
- 收藏 / 标记题目
- 练习中途退出进度保留

## P3：工程与性能

- Room 持久化替代 SharedPreferences JSON（大题库场景）
- 批量导入性能方案
- 单元测试补齐（解析器为核心）
- CI 自动编译 + APK 产出

## Web 壳维护策略

Web 壳作为稳定对照版本，仅做必要维护（安全修复、重大兼容问题）。功能开发以原生版为主，Web 版同步更新。
