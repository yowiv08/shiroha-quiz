适用范围：Shiroha Quiz Web 版、原生 Android Compose 版。

## 一、先说明：推荐格式不等于唯一支持格式

本文档列出的一级、二级、三级格式，是为了让题库导入更稳定，并不是系统唯一支持的格式。

Shiroha Quiz 目前还兼容普通编号、分卷分区、紧凑选项、内嵌答案、答案解析区分离、共用题干 / 材料题、多空填空、表格题库、JSON 题库、跨端备份 JSON/ZIP、图片题等格式。越接近推荐格式，题型、题号、选项和答案越清晰，解析稳定性越高；越接近原始试卷、PDF 抽取文本或网页复制文本，就越可能需要兜底解析或人工清洗。

当前导入规则的核心口径是：**标准格式优先走标准解析，复杂格式只作为兜底候选**。也就是说，结构清楚的题库不应被复杂策略干扰；只有遇到答案解析区、材料题、分卷重复题号、题目和答案分离、排版粘连等情况时，系统才会尝试更多候选策略并选择质量更高的结果。

## 二、当前支持格式与入口

不同文件建议走不同入口，尤其是 Shiroha Quiz 导出的备份文件不要放进普通题库解析区。

| 来源 / 文件                | 推荐入口                       | 说明                                            |
| ---------------------- | -------------------------- | --------------------------------------------- |
| 纯文本 / 复制粘贴 / `.txt`    | 导入题库                       | 按本文档的标准题库格式整理最稳定                              |
| Word `.docx`           | 导入题库                       | 推荐格式之一；原生版可提取内嵌图片并绑定题目                        |
| Excel `.xlsx` / `.xls` | 导入题库                       | 适合一行一题或题目/答案列清晰的表格题库                          |
| 普通题库 JSON              | 导入题库或设置/导出页                | 单题数组、单题库对象可直接识别；多题库集合建议走设置/导出页                |
| 完整备份 JSON / ZIP        | 设置/导出 → 导入配置 / 备份 JSON/ZIP | 用于恢复 Shiroha Quiz 导出的题库、错题、收藏、记录和图片素材         |
| 题目文件 + 答案文件            | 双文件导入                      | 适合试卷和答案分开的情况，建议两个文件都保留章节 / 分区标题               |
| 文字层 PDF                | 导入题库                       | 仅作为辅助支持；扫描件 / 图片型 PDF 不作为主要推荐格式               |
| 图片题                    | `.docx` 或 JSON/ZIP         | 推荐通过 docx 内嵌图片、JSON `images` 字段或原生 ZIP 备份携带图片 |

> 简单理解：普通题库走“导入题库”，Shiroha Quiz 自己导出的备份 JSON/ZIP 走“设置/导出”的备份恢复入口。

## 三、推荐优先级

1. 一级推荐：分区标题 + 普通编号格式。例如“一、单选题”下面写“1. 题干”，这是最接近真实题库、也最方便人工整理的格式。
2. 二级推荐：题号加题型标签格式。例如“1. 单选题：题干”。当题型混杂、分区缺失或需要单题复制时，这种写法更稳定。
3. 三级兼容：分卷分区格式。适合从试卷导出的文档，例如“第一卷”“一、单选题”“二、多选题”“三、判断题”。
4. 其他兼容格式：紧凑选项、内嵌答案、答案解析区分离、共用题干 / 材料题、多空填空、图片题、双文件答案等格式可以导入，但不建议作为人工整理模板。
5. 复杂题库处理建议：如果原题库格式混乱、粘连严重或答案分散，可以先做格式清洗，再导入 Shiroha Quiz；国内用户可优先使用 DeepSeek、GLM / 智谱清言等工具辅助整理，如果可以稳定使用海外模型，也推荐 ChatGPT、Claude。隐私风险主要来自“上传到外部 AI 工具清洗”，不是 Shiroha Quiz 本地导入本身。

## 四、一级推荐格式：分区标题 + 普通编号

说明：这是最接近真实题库的推荐模板。很多题库不会在每一道题前写“单选题 / 多选题 / 判断题”，而是用“一、单选题”“二、多选题”这种分区标题来说明题型。Shiroha Quiz 支持这种写法，并会让分区下的题目继承题型。

如果题库没有分区标题，系统也会根据选项数量、答案形式、题干特征自动判断题型；只是题型混杂、题号重复、题目被单独复制出来时，显式题型会更稳定。

```
一、单选题

1. 题干内容写在这里。
A. 选项一
B. 选项二
C. 选项三
D. 选项四
正确答案：B
解析：这里可以填写解析，也可以不写。

二、多选题

2. 题干内容写在这里。
A. 选项一
B. 选项二
C. 选项三
D. 选项四
正确答案：ACD
解析：这里可以填写解析，也可以不写。

三、判断题

3. 题干内容写在这里。
正确答案：正确
解析：这里可以填写解析，也可以不写。
```

如果担心题型识别不稳定，可以在每道题前增加题型标记。这不是必须写法，但更稳：

```text
1. 单选题：
题干内容写在这里。
A. 选项一
B. 选项二
C. 选项三
D. 选项四
正确答案：B
解析：这里可以填写解析，也可以不写。
```

### 全题型标准模板

下面这一组是最推荐的个人整理模板。它以真实题库最常见的“分区标题 + 普通编号”为主，你可以直接复制后替换题干、选项、答案和解析。

```text
一、单选题

1. 安全帽的主要作用是？
A. 保护头部
B. 装饰作用
C. 增加重量
D. 无实际作用
答案：A
解析：安全帽用于减轻坠落物和碰撞对头部造成的伤害。

二、多选题

2. 雨天驾驶时应注意哪些事项？
A. 降低车速
B. 加大跟车距离
C. 急打方向
D. 紧急制动
答案：AB
解析：雨天路滑，应降低车速并保持安全距离。

三、判断题

3. 国家安全生产方针是“安全第一，预防为主”。
答案：正确
解析：判断题建议使用“正确 / 错误”或“对 / 错”。

四、填空题

4. 依法行政要求行政机关做到____。
答案：合法行政
解析：单空填空题按文本答案保存。

5. 中国四大发明包括____、____、____、____。
答案：造纸术；印刷术；火药；指南针
解析：多空填空题建议按空位顺序用分号分隔。

五、简答题

6. 简述设备启动前需要检查的内容。
参考答案：检查设备状态、核对参数、确认安全防护措施，并进行安全交底。
解析：简答题答案属于文本答案，顿号、逗号、分号不会按选择题答案拆分。
```

说明：

1. 上面这些模板是最贴近常见题库的写法，适合作为个人题库整理标准。
2. 分区标题会让下面的题目继承题型，例如“一、单选题”下面的题目会优先按单选题处理。
3. 如果没有分区标题，题库也可以导入，但题型会依赖选项、答案和题干特征自动判断。
4. 如果担心识别不稳定，可以给单题加题型标记，例如 `1. 单选题：题干`、`2. 多选题：题干`、`3. 判断题：题干`。
5. 填空题、简答题不要整理成 A/B/C/D 选项，答案应保留原始文本。
6. “四、填空题”是一个分区标题，单空和多空都放在这个分区下即可，不需要拆成两个填空题分区。
7. 多空填空题建议用 `；` 分隔每个空的答案；如果原题已经把答案填在括号中，系统也会尝试识别，但人工整理时不建议保留已填答案污染。

### Excel 标准模板

如果题库原本就在表格里，Excel 是很推荐的标准格式。最稳定的写法是“一行一题”，列名清楚写出题干、题型、选项、答案和解析。

```text
题号    题型    题干    A    B    C    D    答案    解析
1    单选题    安全帽的主要作用是    保护头部    装饰作用    增加重量    无实际作用    A    安全帽用于保护头部。
2    多选题    雨天驾驶时应注意哪些事项    降低车速    加大跟车距离    急打方向    紧急制动    AB    雨天路滑，应降低车速并保持安全距离。
3    判断题    国家安全生产方针是“安全第一，预防为主”                    正确    判断题建议使用正确或错误。
4    填空题    依法行政要求行政机关做到____                    合法行政    单空填空题按文本答案保存。
5    简答题    简述设备启动前需要检查的内容                    检查设备状态、核对参数、确认安全防护措施，并进行安全交底。    简答题答案按文本保存。
```

说明：

1. Excel 推荐列名：`题号`、`题型`、`题干`、`A`、`B`、`C`、`D`、`答案`、`解析`。
2. 单选题、多选题填写 A/B/C/D 等选项列。
3. 判断题、填空题、简答题可以不填选项列，只填题干、答案和解析。
4. 如果有章节或分类，可以增加 `章节`、`分区`、`分类` 这类列，尤其适合多个章节题号重复的题库。

### JSON 标准模板

如果你已经有结构化数据，可以使用普通题库 JSON。普通题库 JSON 适合导入题目本身；Shiroha Quiz 导出的完整备份 JSON/ZIP 则应走“设置/导出”的备份恢复入口。

最简单的题目数组：

```json
[
  {
    "type": "single",
    "question": "安全帽的主要作用是？",
    "options": [
      {"key": "A", "text": "保护头部"},
      {"key": "B", "text": "装饰作用"},
      {"key": "C", "text": "增加重量"},
      {"key": "D", "text": "无实际作用"}
    ],
    "answer": ["A"],
    "analysis": "安全帽用于保护头部。"
  },
  {
    "type": "multiple",
    "question": "雨天驾驶时应注意哪些事项？",
    "options": [
      {"key": "A", "text": "降低车速"},
      {"key": "B", "text": "加大跟车距离"},
      {"key": "C", "text": "急打方向"},
      {"key": "D", "text": "紧急制动"}
    ],
    "answer": ["A", "B"],
    "analysis": "雨天路滑，应降低车速并保持安全距离。"
  },
  {
    "type": "blank",
    "question": "中国四大发明包括____、____、____、____。",
    "answer": ["造纸术；印刷术；火药；指南针"],
    "blankAnswers": [["造纸术"], ["印刷术"], ["火药"], ["指南针"]],
    "analysis": "按空位顺序填写答案。"
  },
  {
    "type": "short",
    "question": "简述设备启动前需要检查的内容。",
    "answer": ["检查设备状态、核对参数、确认安全防护措施，并进行安全交底。"],
    "analysis": "简答题答案按文本保存。"
  }
]
```

单题库对象：

```json
{
  "name": "安全知识题库",
  "questions": [
    {
      "type": "single",
      "question": "安全帽的主要作用是？",
      "options": {"A": "保护头部", "B": "装饰作用", "C": "增加重量", "D": "无实际作用"},
      "answer": ["A"],
      "analysis": "安全帽用于保护头部。"
    }
  ]
}
```

说明：

1. 普通题库 JSON 可用于“导入题库”入口。
2. 多题库集合、完整备份 JSON、ZIP 备份应使用“设置/导出 → 导入配置 / 备份 JSON/ZIP”。
3. `type` 常用值可写 `single`、`multiple`、`judge`、`blank`、`short`，也可以使用中文题型列由导入器识别。
4. `options` 可以写成数组，也可以写成对象；对象写法中键名就是选项标号。

## 五、二级推荐格式：普通编号

说明：普通编号格式是很常见的题库写法，Shiroha Quiz 支持导入，不要求每题都写“单选题 / 多选题 / 判断题”。只是没有明确题型标签时，题型会依赖选项数量、答案形式和上下文判断；如果题库本身有“一、单选题”“二、多选题”这类分区标题，识别会更准。多选题建议答案写成 AB、ACD 或 A,B,D。

```
1. 题干内容写在这里。
A. 选项一
B. 选项二
C. 选项三
D. 选项四
答案：A

2. 题干内容写在这里。
A. 选项一
B. 选项二
C. 选项三
D. 选项四
答案：AB
```

## 六、三级兼容格式：分卷分区试卷

说明：分卷分区格式适合从试卷或考试系统导出的文档。系统会根据“一、单选题”“二、多选题”“三、判断题”继承题型上下文。

```
第一卷

一、单选题
题干内容写在这里。【答案：B】
A. 选项一
B. 选项二
C. 选项三
D. 选项四

二、多选题
题干内容写在这里。（ABCD）
A. 选项一
B. 选项二
C. 选项三
D. 选项四

三、判断题
题干内容写在这里。（对）
```

## 七、兼容但不优先推荐的格式

以下格式可以被系统尝试解析，但不建议作为人工整理时的首选模板。

1. 紧凑选项格式。

```
题干内容 A. 选项一 B. 选项二 C. 选项三 D. 选项四 答案：A
```

2. 内嵌答案格式。

```
题干内容【答案：A】
题干内容（ABCD）
题干内容（对）
```

3. 答案与解析分离格式。

```
题目区
1. 题干内容
A. 选项一
B. 选项二

答案解析区
1. 【答案】A
【解析】解析内容
```

4. 无题号但有分区的试卷格式。

```
一、单选题
题干内容【答案：B】
A. 选项一
B. 选项二

二、多选题
题干内容（ABCD）
A. 选项一
B. 选项二
C. 选项三
D. 选项四
```

5. 共用题干 / 材料题格式。

```
根据以下材料，回答 1-2 题：
这里是一段材料文本。

1. 单选题：题干内容写在这里。
A. 选项一
B. 选项二
答案：A

2. 多选题：题干内容写在这里。
A. 选项一
B. 选项二
答案：AB
```

说明：这些格式的识别稳定性依赖文本提取质量。材料题 / 共用题干比普通题更容易把材料、引导语和小题题干混在一起，所以小题建议显式写 `单选题：`、`多选题：` 这类题型标记。如果出现题干、选项、答案粘连，建议先清洗成一级推荐格式。

## 八、判断题写法规范

1. 最推荐写法是单独写答案行，例如“正确答案：正确”或“答案：错误”。
2. 兼容写法是把答案写在题干末尾括号中，例如“题干内容（对）”。
3. 判断题答案建议使用“正确”“错误”“对”“错”“是”“否”“√”“×”。
4. 判断题括号中如果出现 A、B、C、D，不会优先当作判断题答案。它会被视为题干内容或题干污染，除非上下文和选项结构能明确证明它是选择题答案。
5. 不建议用 A、B 表示判断题答案。即使有些试卷把 A 写成正确、B 写成错误，也建议清洗成“正确”“错误”。

<span id="std-09"></span>

## 九、答案与选项规范

1. 答案行推荐写“正确答案：A”“答案：ABCD”“参考答案：正确”。
2. 单选题答案建议写一个字母，例如 A。
3. 多选题答案可以写 AB、ACD、A,B,D 或 A B D。
4. 选项推荐单独成行，并使用 A.、B.、C.、D. 标记。
5. 如果选项被压缩在一行，建议先清洗成分行格式。
6. 解析可选，推荐写在答案下一行，格式为“解析：……”。
7. 填空题、简答题答案属于文本答案，应保留原文，不要因为逗号、顿号、分号把它拆成选择题答案。例如“检查设备、核对参数、进行安全交底。”应作为一整条参考答案保存。

## 十、多空填空题写法规范

多空填空题建议每个空按顺序写清楚，避免把多个空混成一个长答案。

### 支持的填空符号

常见填空符号都可以使用：

```text
____
______
（ ）
( )
（）
()
```

也支持题干中出现“空白处”“横线”“横线上”“括号内”“括号里”“填入”“补全”等提示词来辅助识别。

推荐写法：

```text
四、填空题

4. 中国四大发明包括____、____、____、____。
答案：造纸术；印刷术；火药；指南针
解析：按空位顺序填写答案。
```

括号空也可以写成：

```text
四、填空题

4. 中国四大发明包括（ ）、（ ）、（ ）、（ ）。
答案：造纸术；印刷术；火药；指南针
解析：按空位顺序填写答案。
```

如果原题已经把答案写在括号里，例如：

```text
4. 中国四大发明包括（造纸术）、（印刷术）、（火药）、（指南针）。
答案：造纸术；印刷术；火药；指南针
```

系统也会尝试识别，但人工整理时更推荐改成空括号或横线，避免把括号里的内容误看成题干补充说明。

说明：填空题、简答题属于文本答案，不要为了“看起来整齐”强行改成 A/B/C/D 选择题答案。

<span id="std-11"></span>

## 十一、图片题与富内容规范

图片题推荐使用以下方式导入：

1. 在 `.docx` 中直接插入图片，由原生版提取并绑定到对应题目。
2. 使用 Shiroha Quiz 导出的 JSON/ZIP 备份，图片会通过 `images` 字段或素材文件保留。
3. Web 版旧 JSON 中的 Markdown base64 图片仍会尽量兼容，但不建议人工手写。

不建议手工编写内部图片占位符；如果图片与题目错位，优先回到导入预览中核对。

<span id="std-12"></span>

## 十二、双文件与表格答案补充规范

如果题目文件和答案文件分开，且题库存在分卷、章节或题型分区，建议两个文件都保留相同的分区标题。

推荐写法：

```text
第一章 基础知识
一、单选题
1. A
2. B

第二章 操作规程
一、单选题
1. C
2. D
```

原因是很多试卷每个章节或题型都会从 1 重新编号。如果答案文件只写：

```text
1. A
2. B
1. C
2. D
```

系统虽然会尝试自动匹配，但重复题号越多，越需要依赖顺序或兜底策略，人工核对成本也越高。

Excel / 表格答案同理：如果表格里按章节列答案，请保留章节标题或增加“章节 / 分区”列；每行一题最稳定，答案列使用 A、AB、正确、错误或明确文本答案。

<span id="std-13"></span>

## 十三、不推荐格式

1. 不推荐把多个题目连续写在一段里。
2. 不推荐题干、选项、答案全部粘在一行。
3. 不推荐判断题使用 A/B/C/D 表示对错。
4. 不推荐在题干括号里保留明显答案污染，例如“题干内容（ABCD）”，除非该题明确是多选题。
5. 不推荐把答案集中放在文末且没有题号对应关系。
6. 不推荐把备份 JSON/ZIP 放进普通题库导入区解析。
7. 不推荐把扫描版 PDF 当作主要导入来源。

<span id="std-14"></span>

## 十四、复杂题库清洗建议

如果原题库来自 PDF、扫描件、网页复制、问卷系统或多人合并文档，格式可能会出现粘连、断行、答案错位和题型混杂。建议先进行格式清洗；如使用外部 AI 工具，国内用户可优先选择 DeepSeek、GLM / 智谱清言，能稳定使用海外模型时也推荐 ChatGPT、Claude。隐私风险主要来自“上传到外部 AI 工具清洗”，不是 Shiroha Quiz 本地导入本身。

清洗目标如下。

1. 每道题独立成块。
2. 每道题开头写清题型和题号。
3. 选项统一为 A.、B.、C.、D.。
4. 答案统一为”答案：……”。
5. 判断题统一为”正确”或”错误”。
6. 删除重复说明、页眉页脚、分数说明和无关空行。
7. 保留原始分卷、章节、题型分区标题。
8. 不要解题、不要根据常识推断答案。

## 十五、推荐给智能体的清洗提示词

以下提示词经过两个解析器共同验证，可直接复制使用：

```
请把下面的题库文本整理成 Shiroha Quiz 可稳定导入的标准格式。

要求：
1. 只做格式整理，不要解题，不要改写题意，不要编造题目、选项、答案或解析。
2. 保留所有题目，按原始顺序输出；如果原文有分卷、章节、题型分区，请保留标题。
3. 每道题整理成独立题块，推荐格式为：
题号. 题干
A. 选项
B. 选项
C. 选项
D. 选项
答案：A
解析：原文解析

4. 单选题答案写成：答案：A
5. 多选题答案写成：答案：ABCD
6. 判断题答案写成：答案：正确 或 答案：错误
7. 填空题、简答题答案保留原文，不要拆成选择题答案。
8. 如果原文没有解析，不要编解析，可以省略解析行，或写：解析：
9. 如果答案无法从原文确认，写：答案：【待确认】
10. 最终只输出整理后的题库正文，不要输出说明、分析或 Markdown 代码块。

下面是原始文本：
【把需要清洗的题库粘贴到这里】
```

## 十六、普通题库 JSON 格式导入（v0.7.4+）

如果题库本身已经是结构化 JSON，可以按普通题库 JSON 导入，无需再整理成文本格式。这里说的是“题目内容 JSON”，不是 Shiroha Quiz 的完整备份 JSON/ZIP。

**最简题目数组**：

```json
[
  {"type": "single", "question": "题干", "options": [{"key":"A","text":"选项A"}], "answer": ["A"]}
]
```

**单题库对象**（自动识别题库名）：

```json
{"name": "我的题库", "questions": [...]}
```

**对象选项**（Key 即标号）：

```json
{"options": {"A": "选项内容", "B": "选项内容"}}
```

**多题库数组、完整备份 JSON、ZIP 备份**请到设置/导出页导入。

```text
设置/导出 → 导入配置 / 备份 JSON/ZIP
```

## 十七、快速检查清单

1. 每道题是否能清楚看到题型和题号。
2. 每个选项是否有明确的 A、B、C、D 标记。
3. 每道题是否有独立答案行。
4. 多选题答案是否写成多个字母。
5. 判断题答案是否写成正确或错误。
6. 多空填空题是否按空位顺序写清答案。
7. 如果是图片题，图片是否能在导入预览中正确显示。
8. 是否删除了页眉、页脚、分数说明和无关标题。
9. 备份 JSON/ZIP 是否放到了设置/导出页，而不是普通题库导入区。
10. 如果使用兼容格式，是否在导入预览里检查了题量、题型和答案是否正确。
