# 题库导入标准模板与解析说明

适用范围：Shiroha Quiz Web 版、WebView 壳版、原生 Compose 版。
更新目的：明确推荐标准格式，同时说明系统支持的格式不只推荐格式；用户可以按题库来源选择更合适的整理方式。

## 一、先说明：推荐格式不等于唯一支持格式

本文档列出的一级、二级、三级格式，是为了让题库导入更稳定，并不是系统唯一支持的格式。

Shiroha Quiz 目前还兼容普通编号、分卷分区、紧凑选项、内嵌答案、答案解析区分离等格式。越接近推荐格式，题型、题号、选项和答案越清晰，解析稳定性越高；越接近原始试卷、PDF 抽取文本或网页复制文本，就越可能需要兜底解析或人工清洗。

## 二、推荐优先级

1. 一级推荐：题型加题号格式。每一道题开头写清题型和题号，例如“单选题 1”。这是最稳定的导入格式。
2. 二级推荐：普通编号格式。每一道题使用“1.”或“1、”开头，并在题后写答案。
3. 三级兼容：分卷分区格式。适合从试卷导出的文档，例如“第一卷”“一、单选题”“二、多选题”“三、判断题”。
4. 其他兼容格式：紧凑选项、内嵌答案、答案解析区分离等格式可以导入，但不建议作为人工整理模板。
5. 复杂题库处理建议：如果原题库格式混乱、粘连严重或答案分散，可以先用 LLM 智能体进行数据清洗，再导入 Shiroha Quiz。

## 三、一级推荐格式：题型加题号

说明：这是最推荐的标准模板。每题独立成块，题型、题号、题干、选项、答案、解析都清楚分行。

```
单选题 1
题干内容写在这里。
A. 选项一
B. 选项二
C. 选项三
D. 选项四
正确答案：B
解析：这里可以填写解析，也可以不写。

多选题 2
题干内容写在这里。
A. 选项一
B. 选项二
C. 选项三
D. 选项四
正确答案：ACD
解析：这里可以填写解析，也可以不写。

判断题 3
题干内容写在这里。
正确答案：正确
解析：这里可以填写解析，也可以不写。
```

## 四、二级推荐格式：普通编号

说明：普通编号格式仍然支持，但题型可能需要依赖选项数量、答案形式和上下文判断。多选题建议答案写成 AB、ACD 或 A,B,D。

```
1. 题干内容写在这里。
A. 选项一
B. 选项二
C. 选项三
D. 选项四
答案：A

2. 题干内容写在这里。
A. 选项一
B. 选项二
C. 选项三
D. 选项四
答案：AB
```

## 五、三级兼容格式：分卷分区试卷

说明：分卷分区格式适合从试卷或考试系统导出的文档。系统会根据“一、单选题”“二、多选题”“三、判断题”继承题型上下文。

```
第一卷

一、单选题
题干内容写在这里。【答案：B】
A. 选项一
B. 选项二
C. 选项三
D. 选项四

二、多选题
题干内容写在这里。（ABCD）
A. 选项一
B. 选项二
C. 选项三
D. 选项四

三、判断题
题干内容写在这里。（对）
```

## 六、兼容但不优先推荐的格式

以下格式可以被系统尝试解析，但不建议作为人工整理时的首选模板。

1. 紧凑选项格式。

```
题干内容 A. 选项一 B. 选项二 C. 选项三 D. 选项四 答案：A
```

2. 内嵌答案格式。

```
题干内容【答案：A】
题干内容（ABCD）
题干内容（对）
```

3. 答案与解析分离格式。

```
题目区
1. 题干内容
A. 选项一
B. 选项二

答案解析区
1. 【答案】A
【解析】解析内容
```

4. 无题号但有分区的试卷格式。

```
一、单选题
题干内容【答案：B】
A. 选项一
B. 选项二

二、多选题
题干内容（ABCD）
A. 选项一
B. 选项二
C. 选项三
D. 选项四
```

说明：这些格式的识别稳定性依赖文本提取质量。如果出现题干、选项、答案粘连，建议先清洗成一级推荐格式。

## 七、判断题写法规范

1. 最推荐写法是单独写答案行，例如“正确答案：正确”或“答案：错误”。
2. 兼容写法是把答案写在题干末尾括号中，例如“题干内容（对）”。
3. 判断题答案建议使用“正确”“错误”“对”“错”“是”“否”“√”“×”。
4. 判断题括号中如果出现 A、B、C、D，不会优先当作判断题答案。它会被视为题干内容或题干污染，除非上下文和选项结构能明确证明它是选择题答案。
5. 不建议用 A、B 表示判断题答案。即使有些试卷把 A 写成正确、B 写成错误，也建议清洗成“正确”“错误”。

## 八、答案与选项规范

1. 答案行推荐写“正确答案：A”“答案：ABCD”“参考答案：正确”。
2. 单选题答案建议写一个字母，例如 A。
3. 多选题答案可以写 AB、ACD、A,B,D 或 A B D。
4. 选项推荐单独成行，并使用 A.、B.、C.、D. 标记。
5. 如果选项被压缩在一行，建议先清洗成分行格式。
6. 解析可选，推荐写在答案下一行，格式为“解析：……”。

## 九、不推荐格式

1. 不推荐把多个题目连续写在一段里。
2. 不推荐题干、选项、答案全部粘在一行。
3. 不推荐判断题使用 A/B/C/D 表示对错。
4. 不推荐在题干括号里保留明显答案污染，例如“题干内容（ABCD）”，除非该题明确是多选题。
5. 不推荐把答案集中放在文末且没有题号对应关系。

## 十、复杂题库清洗建议

如果原题库来自 PDF、扫描件、网页复制、问卷系统或多人合并文档，格式可能会出现粘连、断行、答案错位和题型混杂。建议先使用 LLM 智能体进行清洗。

清洗目标如下。

1. 每道题独立成块。
2. 每道题开头写清题型和题号。
3. 选项统一为 A.、B.、C.、D.。
4. 答案统一为“正确答案：……”。
5. 判断题统一为“正确”或“错误”。
6. 删除重复说明、页眉页脚、分数说明和无关空行。

## 十一、推荐给智能体的清洗提示词

以下提示词经过两个解析器共同验证，可直接复制使用：

```
请把下面的待整理文本清洗为适合题库导入的标准格式。
要求如下：
1. 保留所有题目，不要漏题、不要编造题目。
2. 按原始顺序整理，每道题单独成段。
3. 每道题尽量整理为：题号、题干、选项、答案、解析。
4. 单选题答案格式统一为：答案：A
5. 多选题答案格式统一为：答案：ABCD 这种连续大写字母。
6. 判断题答案统一为：答案：正确 或 答案：错误。
7. 填空题、简答题请保留原答案，不要擅自改写。
8. 解析如果原文没有，就不要编造；可以留空或写”解析：”。
9. 遇到无法确认的题干、选项或答案，请标记【待确认】。
10. 最终只输出清洗后的题库正文，不要输出额外解释。

下面是原始文本：
【把需要清洗的题库粘贴到这里】
```

## 十二、快速检查清单

1. 每道题是否能清楚看到题型和题号。
2. 每个选项是否有明确的 A、B、C、D 标记。
3. 每道题是否有独立答案行。
4. 多选题答案是否写成多个字母。
5. 判断题答案是否写成正确或错误。
6. 是否删除了页眉、页脚、分数说明和无关标题。
7. 如果使用兼容格式，是否在导入预览里检查了题量、题型和答案是否正确。
