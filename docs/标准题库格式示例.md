# 题库导入标准模板与解析说明

适用范围：Shiroha Quiz Web 版、WebView 壳版、原生 Compose 版。

> 这份文档是格式模板——给用户参考，按此格式整理题库导入最顺利。
> 更新目的：明确推荐标准格式，同时说明系统支持的格式不只推荐格式；用户可以按题库来源选择更合适的整理方式。

## 一、先说明：推荐格式不等于唯一支持格式

本文档列出的一级、二级、三级格式，是为了让题库导入更稳定，并不是系统唯一支持的格式。

Shiroha Quiz 目前还兼容普通编号、分卷分区、紧凑选项、内嵌答案、答案解析区分离、共用题干 / 材料题等格式。越接近推荐格式，题型、题号、选项和答案越清晰，解析稳定性越高；越接近原始试卷、PDF 抽取文本或网页复制文本，就越可能需要兜底解析或人工清洗。

当前导入规则的核心口径是：**标准格式优先走标准解析，复杂格式只作为兜底候选**。也就是说，结构清楚的题库不应被复杂策略干扰；只有遇到答案解析区、材料题、分卷重复题号、题目和答案分离、排版粘连等情况时，系统才会尝试更多候选策略并选择质量更高的结果。

## 二、推荐优先级

1. 一级推荐：题型加题号格式。每一道题开头写清题型和题号，例如“单选题 1”。这是最稳定的导入格式。
2. 二级推荐：普通编号格式。每一道题使用“1.”或“1、”开头，并在题后写答案。
3. 三级兼容：分卷分区格式。适合从试卷导出的文档，例如“第一卷”“一、单选题”“二、多选题”“三、判断题”。
4. 其他兼容格式：紧凑选项、内嵌答案、答案解析区分离、共用题干 / 材料题等格式可以导入，但不建议作为人工整理模板。
5. 复杂题库处理建议：如果原题库格式混乱、粘连严重或答案分散，可以先做格式清洗，再导入 Shiroha Quiz；国内用户可优先使用 DeepSeek、GLM / 智谱清言等工具辅助整理，如果可以稳定使用海外模型，也推荐 ChatGPT、Claude。隐私风险主要来自“上传到外部 AI 工具清洗”，不是 Shiroha Quiz 本地导入本身。

## 三、一级推荐格式：题型加题号

说明：这是最推荐的标准模板。每题独立成块，题型、题号、题干、选项、答案、解析都清楚分行。

```
单选题 1
题干内容写在这里。
A. 选项一
B. 选项二
C. 选项三
D. 选项四
正确答案：B
解析：这里可以填写解析，也可以不写。

多选题 2
题干内容写在这里。
A. 选项一
B. 选项二
C. 选项三
D. 选项四
正确答案：ACD
解析：这里可以填写解析，也可以不写。

判断题 3
题干内容写在这里。
正确答案：正确
解析：这里可以填写解析，也可以不写。
```

## 四、二级推荐格式：普通编号

说明：普通编号格式仍然支持，但题型可能需要依赖选项数量、答案形式和上下文判断。多选题建议答案写成 AB、ACD 或 A,B,D。

```
1. 题干内容写在这里。
A. 选项一
B. 选项二
C. 选项三
D. 选项四
答案：A

2. 题干内容写在这里。
A. 选项一
B. 选项二
C. 选项三
D. 选项四
答案：AB
```

## 五、三级兼容格式：分卷分区试卷

说明：分卷分区格式适合从试卷或考试系统导出的文档。系统会根据“一、单选题”“二、多选题”“三、判断题”继承题型上下文。

```
第一卷

一、单选题
题干内容写在这里。【答案：B】
A. 选项一
B. 选项二
C. 选项三
D. 选项四

二、多选题
题干内容写在这里。（ABCD）
A. 选项一
B. 选项二
C. 选项三
D. 选项四

三、判断题
题干内容写在这里。（对）
```

## 六、兼容但不优先推荐的格式

以下格式可以被系统尝试解析，但不建议作为人工整理时的首选模板。

1. 紧凑选项格式。

```
题干内容 A. 选项一 B. 选项二 C. 选项三 D. 选项四 答案：A
```

2. 内嵌答案格式。

```
题干内容【答案：A】
题干内容（ABCD）
题干内容（对）
```

3. 答案与解析分离格式。

```
题目区
1. 题干内容
A. 选项一
B. 选项二

答案解析区
1. 【答案】A
【解析】解析内容
```

4. 无题号但有分区的试卷格式。

```
一、单选题
题干内容【答案：B】
A. 选项一
B. 选项二

二、多选题
题干内容（ABCD）
A. 选项一
B. 选项二
C. 选项三
D. 选项四
```

5. 共用题干 / 材料题格式。

```
根据以下材料，回答 1-2 题：
这里是一段材料文本。

1. 题干内容写在这里。
A. 选项一
B. 选项二
答案：A

2. 题干内容写在这里。
A. 选项一
B. 选项二
答案：AB
```

说明：这些格式的识别稳定性依赖文本提取质量。如果出现题干、选项、答案粘连，建议先清洗成一级推荐格式。

## 七、判断题写法规范

1. 最推荐写法是单独写答案行，例如“正确答案：正确”或“答案：错误”。
2. 兼容写法是把答案写在题干末尾括号中，例如“题干内容（对）”。
3. 判断题答案建议使用“正确”“错误”“对”“错”“是”“否”“√”“×”。
4. 判断题括号中如果出现 A、B、C、D，不会优先当作判断题答案。它会被视为题干内容或题干污染，除非上下文和选项结构能明确证明它是选择题答案。
5. 不建议用 A、B 表示判断题答案。即使有些试卷把 A 写成正确、B 写成错误，也建议清洗成“正确”“错误”。

## 八、答案与选项规范

1. 答案行推荐写“正确答案：A”“答案：ABCD”“参考答案：正确”。
2. 单选题答案建议写一个字母，例如 A。
3. 多选题答案可以写 AB、ACD、A,B,D 或 A B D。
4. 选项推荐单独成行，并使用 A.、B.、C.、D. 标记。
5. 如果选项被压缩在一行，建议先清洗成分行格式。
6. 解析可选，推荐写在答案下一行，格式为“解析：……”。
7. 填空题、简答题答案属于文本答案，应保留原文，不要因为逗号、顿号、分号把它拆成选择题答案。例如“检查设备、核对参数、进行安全交底。”应作为一整条参考答案保存。

## 九、双文件与表格答案补充规范

如果题目文件和答案文件分开，且题库存在分卷、章节或题型分区，建议两个文件都保留相同的分区标题。

推荐写法：

```text
第一章 基础知识
一、单选题
1. A
2. B

第二章 操作规程
一、单选题
1. C
2. D
```

原因是很多试卷每个章节或题型都会从 1 重新编号。如果答案文件只写：

```text
1. A
2. B
1. C
2. D
```

系统虽然会尝试自动匹配，但重复题号越多，越需要依赖顺序或兜底策略，人工核对成本也越高。

Excel / 表格答案同理：如果表格里按章节列答案，请保留章节标题或增加“章节 / 分区”列；每行一题最稳定，答案列使用 A、AB、正确、错误或明确文本答案。

## 十、不推荐格式

1. 不推荐把多个题目连续写在一段里。
2. 不推荐题干、选项、答案全部粘在一行。
3. 不推荐判断题使用 A/B/C/D 表示对错。
4. 不推荐在题干括号里保留明显答案污染，例如“题干内容（ABCD）”，除非该题明确是多选题。
5. 不推荐把答案集中放在文末且没有题号对应关系。

## 十一、复杂题库清洗建议

如果原题库来自 PDF、扫描件、网页复制、问卷系统或多人合并文档，格式可能会出现粘连、断行、答案错位和题型混杂。建议先进行格式清洗；如使用外部 AI 工具，国内用户可优先选择 DeepSeek、GLM / 智谱清言，能稳定使用海外模型时也推荐 ChatGPT、Claude。隐私风险主要来自“上传到外部 AI 工具清洗”，不是 Shiroha Quiz 本地导入本身。

清洗目标如下。

1. 每道题独立成块。
2. 每道题开头写清题型和题号。
3. 选项统一为 A.、B.、C.、D.。
4. 答案统一为”答案：……”。
5. 判断题统一为”正确”或”错误”。
6. 删除重复说明、页眉页脚、分数说明和无关空行。
7. 保留原始分卷、章节、题型分区标题。
8. 不要解题、不要根据常识推断答案。

## 十二、推荐给智能体的清洗提示词

以下提示词经过两个解析器共同验证，可直接复制使用：

```
请把下面的题库文本整理成 Shiroha Quiz 可稳定导入的标准格式。

要求：
1. 只做格式整理，不要解题，不要改写题意，不要编造题目、选项、答案或解析。
2. 保留所有题目，按原始顺序输出；如果原文有分卷、章节、题型分区，请保留标题。
3. 每道题整理成独立题块，推荐格式为：
题号. 题干
A. 选项
B. 选项
C. 选项
D. 选项
答案：A
解析：原文解析

4. 单选题答案写成：答案：A
5. 多选题答案写成：答案：ABCD
6. 判断题答案写成：答案：正确 或 答案：错误
7. 填空题、简答题答案保留原文，不要拆成选择题答案。
8. 如果原文没有解析，不要编解析，可以省略解析行，或写：解析：
9. 如果答案无法从原文确认，写：答案：【待确认】
10. 最终只输出整理后的题库正文，不要输出说明、分析或 Markdown 代码块。

下面是原始文本：
【把需要清洗的题库粘贴到这里】
```

## 十三、JSON 格式导入（v0.7.4+）

如果题库已经是 JSON 格式，可直接导入，无需按文本格式整理。

**最简题目数组**：
```json
[
  {"type": "single", "question": "题干", "options": [{"key":"A","text":"选项A"}], "answer": ["A"]}
]
```

**单题库对象**（自动识别题库名）：
```json
{"name": "我的题库", "questions": [...]}
```

**对象选项**（Key 即标号）：
```json
{"options": {"A": "选项内容", "B": "选项内容"}}
```

**多题库数组**请到设置/导出页导入。

## 十四、快速检查清单

1. 每道题是否能清楚看到题型和题号。
2. 每个选项是否有明确的 A、B、C、D 标记。
3. 每道题是否有独立答案行。
4. 多选题答案是否写成多个字母。
5. 判断题答案是否写成正确或错误。
6. 是否删除了页眉、页脚、分数说明和无关标题。
7. 如果使用兼容格式，是否在导入预览里检查了题量、题型和答案是否正确。
