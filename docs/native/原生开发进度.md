# Shiroha Quiz 原生 Compose 开发进度

## 项目概述

Shiroha Quiz 是一个自导入题库的刷题/考试 App，采用双入口架构：

| 入口         | 技术栈                                  | 包名                      | 状态   |
| ---------- | ------------------------------------ | ----------------------- | ---- |
| Web 壳子     | WebView 加载 HTML/CSS/JS               | `com.yiqiu.shirohaquiz` | 功能完整 |
| 原生 Compose | Kotlin + Jetpack Compose + Material3 | `com.reqir.shirohaquiz` | 开发中  |

## 原生页面（12 个）

| 页面       | 文件                              | 功能描述                                            |
| ---------- | ------------------------------- | --------------------------------------------------- |
| 主页       | `HomeScreen.kt`                 | 题库概览卡片、题库列表、快速入口                             |
| 导入页     | `ImportScreen.kt`               | 文件选择、文本解析、双文件导入、识别预览、后台处理                  |
| 练习页     | `PracticeScreen.kt`             | 题卡展示、选项交互、答案提交、解析展示、随机/顺序组题、紧凑设置面板      |
| 考试页     | `ExamScreen.kt`                 | 题量设置、计时器、答题卡、未答交卷提醒、得分统计、明细报告             |
| 题库列表   | `BankListScreen.kt`             | 题库管理、重命名、删除                                  |
| 题库详情   | `BankDetailScreen.kt`           | 题库内题目列表查看、二次核对                                |
| 题库复查   | `BankReviewScreen.kt`           | 题目逐题检查与修正                                     |
| 错题本     | `WrongBookScreen.kt`            | 错题汇总与复习                                        |
| 学习记录   | `RecordsScreen.kt`              | 一轮一条记录、逐题复盘                                  |
| 我的       | `MeScreen.kt`                   | 设置、数据导入/导出/备份、关于                              |
| 关于       | `AboutScreen.kt`                | 版本信息与应用说明                                     |
| 记录详情   | `RecordDetailScreen.kt`         | 单轮练习/考试记录逐题复盘                                |
| 导入格式   | `StandardImportFormatScreen.kt` | 标准导入格式说明与 AI 清洗指引                            |

## 架构

```
MainActivity (入口路由)
  → ShirohaAppShell (底部导航栏 + 多页面)
    → HomeScreen / ImportScreen / PracticeScreen / ExamScreen / BankListScreen /
      BankDetailScreen / BankReviewScreen / WrongBookScreen / RecordsScreen /
      MeScreen / AboutScreen / RecordDetailScreen / StandardImportFormatScreen
```

- 所有页面共享 `QuizRepository` 单例状态
- 页面间通过回调导航（`onGoImport`、`onGoPractice` 等）
- 数据持久化：SharedPreferences（JSON 序列化）
