# Shiroha Quiz UI 素材包 v1

本包按 WebApp 使用场景分类整理，文件均为 PNG 方图素材。

## 分类与用途

- 
- `01_主页欢迎/home_welcome_avatar.png`：首页欢迎区、空白首页、启动后的轻量引导。
- `02_空状态/empty_state_no_question_bank.png`：无题库、无题目、无搜索结果等空状态。
- `03_导入题库/import_question_bank_folder.png`：导入题库入口、首次导入引导、导入成功前提示。
- `04_刷题学习/quiz_in_progress_study.png`：刷题页、练习模式、学习中状态。
- `05_错题复习/wrong_question_review_clipboard.png`：错题本、错题复习、错误反馈、待修正题目。
- `06_加载等待/loading_waiting_hourglass.png`：加载中、解析中、导入处理中、题库统计中。
- `07_休息睡眠/rest_sleep_pillow.png`：休息提示、完成练习后、夜间/轻松状态。
- `08_思考提示/thinking_state_question.png`：解析策略选择、确认弹窗、帮助说明、题目识别提示。

## 建议使用方式

- 首页/空状态主图建议宽度控制在 `120px–180px`。
- 弹窗或卡片内插图建议宽度控制在 `96px–140px`。
- 背景较浅，适合搭配白色卡片、淡紫背景和蓝紫主色按钮。
- 若用于 Android WebView，建议先放入 `app/src/main/assets/web/assets/illustrations/` 或类似目录，再由前端 CSS/JS 引用。
