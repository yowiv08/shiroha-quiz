智能导入

支持常见 TXT 排版、Word .docx 题库、文字型 PDF 文本层导入、问卷星式“正确答案”标记、题后“答案：D”、JSON 里的 A/B/C/D 字段、填空题/简答题参考答案，以及题目文件与答案文件分离导入。

考试模式

支持考试名称、题量、限时、题型分值、及格线、自动交卷，交卷后生成得分、正确率、题型统计和明细报告。

记录与错题

练习和考试都会生成明细记录，答错自动进入错题本并累计错误次数，可按掌握状态复习。

单文件智能导入

Dual File Import

双文件导入：题目文件 + 答案文件

适用于“一个文件只写题目，另一个文件只写答案”的试卷/答案卷。题目文件支持 .docx / .txt / .json / 文字型 .pdf；答案文件支持 .docx / .txt / 文字型 .pdf。文字型 PDF 使用本地文本提取能力，不依赖 OCR。

当前重点支持的题库写法

本版重点增强 Word/TXT/PDF/JSON 智能识别，暂不做 Excel。PDF 先支持文字型 PDF：能复制文字的 PDF 可直接提取文本；扫描版/图片版 PDF 暂不做 OCR。PDF 采用混合加载：优先使用 libs 里的本地 PDF.js，缺失时联网加载 CDN PDF.js，仍失败时使用轻量提取器兜底。可识别：题型标记、题号、A-D/A-G 选项、数字选项、无字母选项、答案写在选项后、答案单独一行、答案写在题号后、解析、判断题对错、多选题连续答案、同一行多个选项、填空题、简答题。
