Dual File Import

双文件导入：题目文件 + 答案文件

适用于“一个文件只写题目，另一个文件只写答案”的试卷/答案卷。题目文件支持 .docx / .txt / .json / 文字型 .pdf；答案文件支持 .docx / .txt / 文字型 .pdf。文字型 PDF 使用本地文本提取能力，不依赖 OCR。
