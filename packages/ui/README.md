# ui — 设计令牌

原生 Compose 版设计系统。Web 版视觉对齐参考。

## 间距 (ShirohaSpacing)

| 令牌 | 值 |
|------|----|
| `Xs` | 4dp |
| `Sm` | 8dp |
| `Md` | 12dp |
| `Lg` | 16dp |
| `Xl` | 24dp |

## 颜色 (ShirohaColors)

| 令牌 | 用途 |
|------|------|
| `BrandPrimarySoft` | 主色浅底 |
| `CardSoft` | 卡片背景 |
| `CardMuted` | 弱化背景 |
| `LineSoft` | 分割线 |
| `LineSelected` | 选中边框 |
| `TextSecondary` | 次要文字 |

## 圆角 (ShirohaRadius)

| 令牌 | 值 |
|------|----|
| `Lg` | 20dp |
| `Pill` | 999dp |

## 核心组件

| 组件 | 说明 |
|------|------|
| `GlassCard` | 毛玻璃卡片，含入场动画 |
| `ActionPillButton` | 胶囊按钮 |
| `IllustrationHeroCard` | 插画头图 |
| `MetricGlassCard` | 数据指标卡 |
| `NoticeCard` | 提示卡片 |
